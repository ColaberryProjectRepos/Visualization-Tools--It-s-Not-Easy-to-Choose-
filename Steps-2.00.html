What Advanced Analytics Capabilities do the tools have? 2

        Power BI

• Predictive analytics:Using Azure Machine Learning Studio, users can quickly create predictive models by dragging, dropping, and connecting data modules. Power BI then allows users to visualize the results of their machine learning algorithm.

• Data visualizations :Using the R Showcase, you’ll be able to apply complex algorithms, visualizations, scripts, and more with just a click. You don’t even need to know anything about R to use the R custom visuals

• Data analysis expressions : Data analysis expressions (DAX )helps you create new information from data that is already in your model.
