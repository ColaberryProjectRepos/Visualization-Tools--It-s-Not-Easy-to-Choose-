What are the different Visual Capabilities of Power BI, Qlik and Tableau? 1

Tableau

•  Most attractive and intuitive visualization tool, according to Gartner’s 2017 “Magic Quadrant for BI and Analytics Platforms.                                                                    

•  In-memory data processing for quick refreshes of visualization during data exploration.                                         

•  Larger data sets require direct calls to data source

                                            



            Qlik 

•   In-memory data engine lets users build interactive graphics.                                                

•   Holding data in memory allows graphics to update quickly while users explore data sets visually.

•   Robust data engine allows multiple sources of data to be combined and visualized



                                        


            Power BI

• Custom visuals: Used in Power BI reports.Custom visuals is wriiten in TypeScript, a superset of Javascript and storyteller

• Embedding : The Power BI service (SaaS) and the Power BI Embedded service in Azure (PaaS) have APIs for embedding your dashboards and reports.
